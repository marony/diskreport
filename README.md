# ディスク容量可視化ツール(scalafx on JavaFX8)

　scalafxもJavaFX8も全然知らないので練習で作ってみました。

    Scalaが入っている環境で起動
    $ activator run
    
    Windows用のインストーラ(msi)作成
    $ activator windows:packageBin
    
    Mac用のインストーラ(DMG)作成
    $ activator universal:packageOsxDmg
